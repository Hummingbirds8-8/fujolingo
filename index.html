APIキーをお持ちでない場合、以下のプリセットストーリーを選んでアプリの全機能を試すことができます。

🔑 Gemini API 設定

ストーリーの新規生成には Gemini API キーが必要です。Vercelでホストし、環境変数 GEMINI_API_KEY を設定している場合は、空欄のままで動作します。キーを入力した場合は優先的に使用されます。

🎭 カップリング管理

お好みの攻め・受けのペアを複数登録できます。生成タブから切り替えて、それぞれのカップリングごとのお話（単発/連載）を楽しむことができます。
